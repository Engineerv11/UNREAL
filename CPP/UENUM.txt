UENUM(BlueprintType)
namespace A
{
    enum EVehicle
    {
        Aircraft,
        Car,
        Ship
    }
}

将枚举暴露到蓝图

使用BlueprintType，在蓝图中可以创建此枚举类型的变量

////////////////////////////////////////////////////////////////////////////////

UPROPERTY(EditAnywhere,BlueprintReadWrite,Category="VehicleInfo")
TEnumAsByte<EVehicle> Vehicle;

在蓝图面板中显示下拉框，选择载具类型，Aircraft，Car，Ship
